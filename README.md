# Private Workspace Repository

This repository contains my personal OpenClaw workspace configuration and memories.

## ⚠️ Security Notice

This repo is **private** and contains sensitive configuration files. 

### API Keys Protection
- All API keys and secrets are protected by `.gitignore`
- Never commit files containing:
  - API keys
  - Access tokens
  - Private keys
  - Environment variables with secrets

## Structure

- `AGENTS.md` - Agent configuration and rules
- `SOUL.md` - Agent persona and identity
- `USER.md` - User preferences
- `TOOLS.md` - Local tool configurations
- `MEMORY.md` - Long-term memory (only in private contexts)
- `memory/` - Daily memory logs
- `HEARTBEAT.md` - Periodic task definitions
- `IDENTITY.md` - Identity configuration

## Setup

1. Clone this repository
2. Create a `.env` file with your API keys (not tracked by git)
3. Install OpenClaw and configure
